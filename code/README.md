# No-IDE Arduino

Works with Debian 11.

Install with
```bash
git clone --recurse-submodules https://github.com/so-rose/dtu-solar-cooker.git
```



# Setup

## Setup (Debian)
Install `arduino-mk` using:
```bash
sudo apt install arduino-mk
```
